 项目. 基于 `vue` 和 `element-ui` 构建.

## 部分特点
* 列表页类型页面的 Table 支持, 动态显示, 隐藏, 排序列.
* element-ui 组件的按需导出(包括CSS样式)等.

## 构建 & 运行

1. 
```
git clone https://github.com/0xChen/lore-blog-admin
```

2.
```
npm install --registry=https://registry.npm.taobao.org
```

### 运行
```
npm run serve
```
### 构建
```
npm run build
```
